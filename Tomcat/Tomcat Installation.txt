Tomcat INSTALLATION:
1. Create a New Ec2 instance with Amazon Linux
2. Tag: Tomcat_server
3. Security Group: All traffic: 0.0.0.0/0
4. Download new key pair and save it

Open the Ec2 instance at Mobaterm.
1. $yum install java-1.8* -y
2. $java -version
3. Set Java path
	$find /usr/lib/jvm/java-1.8* | head -n 3
	$vi /.bash_profile
	JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-<Java version which seen in the above output>
	export JAVA_HOME
	PATH=$PATH:$JAVA_HOME
	$source ./bash_profile
	$echo $JAVA_HOME

4. Tomcat Download(https://tomcat.apache.org/download-80.cgi)
	$cd /opt
	$wget https://dlcdn.apache.org/tomcat/tomcat-8/v8.5.83/bin/apache-tomcat-8.5.83.tar.gz
	$tar -xvzf /opt/apache-tomcat-8.5.83.tar.gz
	$chmod +x /opt/apache-tomcat-8.5.83/bin/startup.sh 
	$chmod +x /opt/apache-tomcat-8.5.83/bin/shutdown.sh 
	$ln -s /opt/apache-tomcat-8.5.83/bin/startup.sh /usr/local/bin/tomcatup
	$ln -s /opt/apache-tomcat-8.5.83/bin/shutdown.sh /usr/local/bin/tomcatdown
	$tomcatdown
	$tomcatup

5. Connect with server using http://<Public_IP>:8080

6. Change Connector Port
	$vi /opt/apache-tomcat-8.5.83/conf/server.xml
	update the port as "8090" & save.
	Restart the Tomcat server to check the changes.
	$tomcatdown
	$tomcatup

7. Connect with server using http://<Public_IP>:8090
8. Tomcat application doesnt allow to login from browser. changing a default parameter in context.xml does address this issue
	$ find / -name context.xml
	comment () Value ClassName field on files which are under webapp directory. <!--   -->
	$tomcatdown
	$tomcatup

9. Update users information in tomcat-users.xml
	$vi /opt/apache-tomcat-8.5.83/conf/tomcat-users.xml
	 <role rolename="manager-gui"/>
	 <role rolename="manager-script"/>
	 <role rolename="manager-jmx"/>
 	 <role rolename="manager-status"/>
 	 <user username="admin" password="admin" roles="manager-gui, manager-script, manager-jmx, manager-status"/>
 	 <user username="deployer" password="deployer" roles="manager-script"/>
 	 <user username="tomcat" password="s3cret" roles="manager-gui"/>
	$tomcatdown
	$tomcatup
